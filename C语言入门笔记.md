# · C语言 入门笔记

---

> C语言第一个程序“Hello word”

#### · Hello word

```c
#include <stdio.h>
 int main() 
 {
  printf("Hello World");
  return 0;
 }
```

####  每行解释

1. `#include <stdio.h>`
   
   - 引入库：`Standard Input Output.Header`
   
2. `int main() { }`
   
   - 程序的入口函数
   - 程序从这里开始执行
   - 程序返回值为`int`型整数
   
3. `printf("Hello World");`
   - 向控制台中输出`Hello World`
   
4. `return 0;`
   - 返回值为0
